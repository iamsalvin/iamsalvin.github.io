Watch Store Website

Project Date June 15th, 2023
Role Developer, Designer
Client Watch Shop
Visit https://watchstore.com

This project involved creating a user-friendly website for a watch store, allowing customers to browse and book products online. The website features a sleek design and intuitive interface, making it easy for users to find and purchase their desired watches.

The website integrates various functionalities, including product listings, booking system, and secure payment options. This project showcases my skills in web development and design, providing a seamless shopping experience for customers.
